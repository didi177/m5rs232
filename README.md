# m5rs232

M5 Atom lite                            https://shop.m5stack.com/products/atom-lite-esp32-development-kit
with ATOMIC RS232 Base W/O Atom lite    https://shop.m5stack.com/products/atomic-rs232-base-w-o-atom-lite

description:
